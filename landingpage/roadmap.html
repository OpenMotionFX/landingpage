Project Status

No Vaporware.
Just Milestones.

We do not sell pre-orders for hardware that doesn't exist. This project is heavily in active development by a core team of engineers and filmmakers. Follow the real-time repository commits, or track our macro-level build phases below.

Current Focus Phase 0

Bench Prototyping & Architecture

Solidifying the core mathematical engines and local-first protocol constraints on breadboards before cutting aluminum or printing heavy plastic.

  • ESP32-S3 firmware foundation & OTA updates
  • UDP beacon microsecond synchronization testing
  • JSON payload structure for `.omfx` protocol
Exit condition: A motor spins exactly 10,000 steps via Wi-Fi command with zero skipped pulses.
Upcoming Phase 1

Pan/Tilt Module & Conductor v0.1

Bringing the first physical product online. The Pan/Tilt head alone allows filmmakers to achieve massive scale and twinning VFX techniques.

  • Design cycloidal gearboxes for 3D printing
  • Build the initial Vue.js/Electron Conductor desktop app
  • Implement keyframe Bezier math interpolation
Exit condition: A complete, repeatable 2-axis nodal pan shot filmed on an iPhone.
Planned Phase 2

Linear Tracking (Slider & Turntable)

Adding translation to the ecosystem. Integrating standard GT2 belts and aluminum extrusions into the firmware calculation path.

  • Homing and hard-stop automatic calibration limits
  • VFX Blender Add-on for X-axis spatial mapping
  • Turntable gear-reduction chassis
Planned Phase 3

Ecosystem (Lighting & Sync Box)

Bringing the rest of the set into the deterministic loop. Matching lighting cues with physical camera position.

  • High-CRI COB LED PWM dimming drivers
  • Timecode injection via the Sync Box
  • Triggering DSLR/Mirrorless shutters via 2.5mm jack
Planned Phase 4

1.0 Public Release

Locking the API, finalizing all 3D print STLs, publishing the ultimate step-by-step assembly guides, and opening the floodgates to the maker community.
